== MailConfig

|===
^|Name | Type ^| Description

|`hostname`
|`String`
|+++

|`port`
|`Number`
|+++
Set the port of the smtp server.+++

|`username`
|`String`
|+++
Set the username for the login.+++

|`password`
|`String`
|+++
Set the password for the login.+++

|`ssl`
|`Boolean`
|+++
Set the sslOnConnect mode for the connection.+++
